# raw_sources/product/

存放**原始产品文档**的占位目录。

## 用法
- 每个产品建一个子目录：`raw_sources/product/[产品名]/`
- 投喂该产品的原始文档（设计说明、实现笔记、私有行为描述等）
- ingest 后，编译结果写入 `wiki/compiled/product_notes/[产品名]/`

## 权限
- 本目录归属 `raw_sources/`，LLM **只读**，仅由人投喂。

## 约束
- 产品私有行为在编译页面中必须标注 **⚠️ 产品私有，不代表 3GPP 标准**。
- 当前为初始化占位，尚未创建任何具体产品目录。
