# TS38.305 — Stage 2 functional specification of UE positioning in NG-RAN

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.305）。
> 原始文档：[[38.305_UE_Positioning_j10]]（`raw_sources/specs/TS38.305_UE_Positioning/38.305_UE_Positioning_j10.md`）

- **职责**：NG-RAN UE 定位 Stage 2——架构、信令、过程、定位方法。

## 架构与信令（§5–§6）
- 核心网网元 **LMF**（Location Management Function）；接口：UE↔LMF 经 **LPP**（TS37.355），gNB↔LMF 经 **NRPPa**（TS38.455，经 NGAP 透传 [[TS38.413_NGAP]]）。

## 定位方法（§8）
| 方法 | 说明 |
|------|------|
| **DL-TDOA** | 下行到达时间差（基于 PRS） |
| **DL-AoD** | 下行离开角 |
| **UL-TDOA** | 上行到达时间差（基于 SRS） |
| **UL-AoA** | 上行到达角 |
| **Multi-RTT** | 多站往返时间 |
| E-CID / RAT-independent | 增强小区 ID、GNSS、传感器等 |

- 关键信号：**DL PRS**（定位参考信号）、**UL SRS for positioning**。
- 测量量（RSTD、Rx-Tx 时间差、AoA/AoD 等）定义见 [[TS38.215_PHY_Measurements]]。
