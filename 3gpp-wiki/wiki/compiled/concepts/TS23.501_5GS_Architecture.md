# TS23.501 — System architecture for the 5G System (5GS); Stage 2

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS23.501）。
> 原始文档：[[23.501_5GS_Architecture_j70]]（`raw_sources/specs/TS23.501_5GS_Architecture/23.501_5GS_Architecture_j70.md`）

- **职责**：5G 系统架构 Stage 2（SA WG2，核心网）。接入网研发的**核心网参考上下文**。

## 架构模型（§4）
- **SBA**（服务化架构）：控制面 NF 经服务化接口交互。
- 关键 NF（§6）：**AMF**（接入移动管理）、**SMF**（会话管理）、**UPF**（用户面）、**AUSF/UDM**（认证/数据）、**PCF**（策略）、**NRF/NSSF**（发现/切片选择）。
- UE—(R)AN—UPF—DN 用户面；UE—AMF NAS 控制面。

|------|--------|------|
| 注册与连接管理 | §5.3 | RM/CM 状态对应 NAS（[[TS24.501_NAS_5GS]]） |

| 双连接/多连接 | §5.11 | 对接 [[TS37.340_Multi_Connectivity]] |
| 网络切片 | §5.15 | S-NSSAI |

## 关键概念（接入网常用）
- **5QI**：标准化 QoS 标识（资源类型、优先级、时延预算、误包率）。
- **PDU Session / QoS Flow / DRB** 三级映射：5GC 建 QoS flow → SDAP 映射到 DRB。
