# TS38.423 — NG-RAN; Xn Application Protocol (XnAP)

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.423）。
> 原始文档：[[38.423_XnAP_j20]]（`raw_sources/specs/TS38.423_XnAP/38.423_XnAP_j20.md`）

- **职责**：**Xn 接口**（gNB ↔ gNB）控制面应用协议；支撑 Xn 切换、双连接、负载管理。基于 SCTP。

## 主要过程组（§8）
| 组 | clause | 关键 EP |
|----|--------|---------|
| 基本移动性 | §8.2 | **Handover Preparation**、SN Status Transfer、UE Context Release、Retrieve UE Context、Path Switch（经 NG 配合） |
| **双连接 DC** | §8.3 | **S-Node Addition/Modification/Release**（SgNB 添加/修改/释放）、Activity Notification |
| 全局过程 | §8.4 | Xn Setup、NG-RAN Node Configuration Update、Cell Activation、Resource Status（负载）、Error Indication |
| IAB | §8.5 | IAB 相关 |
| L1/L2 触发移动性 LTM | §8.6 | LTM 相关 |

## 关键概念
- Xn handover：源 gNB ↔ 目标 gNB 直接协调（无需经核心网），失败回退 NG handover。
- DC：MN ↔ SN 经 Xn 协调 SCG 配置（配合 [[TS37.340_Multi_Connectivity]]）。
