# TS38.133 — Requirements for support of radio resource management

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.133）。
> 原始文档：[[38.133_RRM_j40]]（`raw_sources/specs/TS38.133_RRM/38.133_RRM_j40.md`）

- **职责**：**RRM 性能要求**（RAN4）——移动性时延、定时精度、测量精度/上报的最低指标。UE 实现与一致性测试的依据。

## 主要章节
| clause | 内容 |
|--------|------|
| §4 | RRC_IDLE 移动性（小区选择/重选时延、测量要求） |
| §5 | RRC_INACTIVE 移动性 |
| §6 | **RRC_CONNECTED 移动性**（handover 时延/中断、PSCell 变更等） |
| §7 | **Timing**（发送定时、TA 精度、UE 定时误差 Te） |
| §8 | 信令特性 |
| §9 | **测量过程**（测量周期、要求） |
| §10+ | **测量精度**（SS-RSRP/RSRQ/SINR 绝对/相对精度等） |

## 关键点
- 定义"多久内完成 / 多准"的下界：如小区识别时延、测量上报周期、HO 中断时间、RSRP 精度。
- 与测量量定义（[[TS38.215_PHY_Measurements]]）配套：215 定义"是什么"，133 定义"要多准/多快"。
