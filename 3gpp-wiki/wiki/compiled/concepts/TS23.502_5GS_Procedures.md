# TS23.502 — Procedures for the 5G System (5GS); Stage 2

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS23.502）。
> 原始文档：[[23.502_5GS_Procedures_j70]]（`raw_sources/specs/TS23.502_5GS_Procedures/23.502_5GS_Procedures_j70.md`）

- **职责**：5G 系统流程 Stage 2（SA WG2）。是 [[TS23.501_5GS_Architecture]] 的流程展开。

## 系统流程（§4）
| 流程 | 接入网相关点 |
|------|------|
| **Registration** | UE 注册；经 RAN 转发 Initial UE Message（[[TS38.413_NGAP]]） |
| **Service Request** | UE/网络触发；建立 N2/N3、AN 资源 |
| **PDU Session Establishment** | SMF 建会话；下发 QoS flow → RAN 建 DRB |
| PDU Session Modification/Release | QoS 更新/释放 |
| **Handover**（Xn/N2） | Xn HO（[[TS38.423_XnAP]]）/ N2 HO（经 AMF），路径切换 |
| Paging / AN Release | CN 寻呼、RAN 连接释放 |

## NF 服务流程（§5）
基于 SBA 的 NF service（如 Namf_Communication、Nsmf_PDUSession 等）。

## 关键点
- 接入网在这些流程里承担 **N2（NGAP）信令** 与 **N3（GTP-U）用户面** 角色。
- PDU Session 建立中 QoS flow 参数下发给 RAN，触发 [[TS37.324_SDAP]]/[[TS38.331_RRC]] 配置 DRB。
