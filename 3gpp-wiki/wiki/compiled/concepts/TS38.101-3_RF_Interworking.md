# TS38.101-3 — UE RF; Part 3: Range 1 and Range 2 Interworking operation for non-standalone

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.101-3）。
> 原始文档：[[38.101-3_RF_Requirements_Interworking_j50]]（`raw_sources/specs/TS38.101-3_RF_Interworking/...`）

- **职责**：**UE 射频要求——多频段/多 RAT 互操作**（RAN4）：EN-DC、NR-DC、NR-CA、FR1+FR2 同时工作。

## 覆盖场景
| 场景 | 说明 |
|------|------|
| **EN-DC** | LTE + NR 双连接（NSA），LTE 锚点 + NR 辅载波 |
| **NR-DC** | NR(FR1) + NR(FR2) 双连接 |
| **NR CA** | NR 载波聚合（带内/带间） |
| FR1+FR2 | 同时工作的发射/接收要求 |

## 关键概念
- **配置（configuration）**：列出支持的频段组合（band combination）。
- **MSD（Maximum Sensitivity Degradation）**：多频段同时工作时的灵敏度恶化预算（互调、谐波落入接收带）。
- 总发射功率、功率分配、A-MPR（针对组合场景）。
