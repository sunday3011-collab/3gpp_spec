# TS38.314 — NR; Layer 2 measurements

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.314）。
> 原始文档：[[38.314_L2_Measurements_j00]]（`raw_sources/specs/TS38.314_L2_Measurements/38.314_L2_Measurements_j00.md`）

- **职责**：定义 L2（MAC/RLC/PDCP 层）测量量——供网络优化、MDT/SON、调度评估使用。

## 测量量（§4）
| 测量 | 执行方 | 说明 |
|------|--------|------|
| gNB 测量（§4.2） | gNB | 适用所有部署场景的 L2 量（如 PRB 利用率、UL/DL 数据量、激活 UE 数等） |
| **Packet delay**（§4.3.1） | UE | 上行/下行包时延（D2U/Uu 时延），用于时延 KPI |

## 关键点
- 与 PHY 测量（[[TS38.215_PHY_Measurements]]）互补：215 是 L1 无线质量量，314 是 L2 吞吐/时延/负载量。
- 上报机制经 RRC（[[TS38.331_RRC]]）配置；部分用于 MDT。
