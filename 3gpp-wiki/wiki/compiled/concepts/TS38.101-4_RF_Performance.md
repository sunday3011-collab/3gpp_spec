# TS38.101-4 — UE RF; Part 4: Performance requirements

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.101-4）。
> 原始文档：[[38.101-4_RF_Performance_j22]]（`raw_sources/specs/TS38.101-4_RF_Performance/...`）

- **职责**：**UE 性能要求**（RAN4）——解调性能与 CSI 上报准确性的最低指标（区别于 38.101-1/2 的发射/接收 RF 指标）。

## 主要内容
| 类别 | 要点 |
|------|------|
| **PDSCH 解调** | 各信道模型/调制/MIMO 下的吞吐量门限（通常 70% 最大吞吐） |
| **PDCCH 解调** | 控制信道解调要求 |
| **PBCH / 同步** | 接收性能 |
| **CSI 上报** | CQI/PMI/RI 的报告准确性要求 |

## 关键点
- 基于传播条件（衰落信道模型 TDL/CDL、多普勒）、天线配置定义性能门限。
- 是一致性测试 TS38.521-4 对应的 RF 性能依据。
