# TS38.473 — NG-RAN; F1 Application Protocol (F1AP)

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.473）。
> 原始文档：[[38.473_F1AP_j20]]（`raw_sources/specs/TS38.473_F1AP/38.473_F1AP_j20.md`）

- **职责**：**F1 接口**（gNB-CU ↔ gNB-DU）控制面应用协议（F1-C）；支撑 CU/DU 切分下的 UE 上下文、RRC 透传、系统信息、寻呼。基于 SCTP。

## 主要过程组（§8）
| 组 | clause | 关键 EP |
|----|--------|---------|
| 接口管理 | §8.2 | **F1 Setup**、gNB-CU/DU Configuration Update、Reset、Error Indication |
| **UE 上下文管理** | §8.3 | UE Context Setup/Modification/Release（含承载、SCell、DRB 配置） |
| **RRC 消息透传** | §8.4 | DL/UL RRC Message Transfer（CU 产生 RRC，DU 经空口收发） |
| 系统信息 | §8.6 | DU 广播、CU 提供其他 SI |
| 寻呼 | §8.7 | Paging（CU→DU） |
| 定位 | §8.13 | Positioning 相关 |
| 其他 | §8.5/8.9–8.17 | 告警、无线信息、IAB、参考时间、MBS、QMC 等 |

## 关键概念
- **职责切分**：RRC/PDCP 在 CU，RLC/MAC/PHY 在 DU；F1AP 传 UE 上下文与 RRC 容器。
- 标识：gNB-CU UE F1AP ID / gNB-DU UE F1AP ID。
