# TS38.212 — Multiplexing and channel coding

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.212）。
> 原始文档：[[38.212_PHY-Coding_j30]]（`raw_sources/specs/TS38.212_Coding/38.212_PHY-Coding_j30.md`）

- **层**：PHY（RAN WG1）
- **职责**：传输信道与控制信息的复用、信道编码、速率匹配。

## 通用编码流程（§5）
1. CRC 计算（§5.1）
2. 码块分段 + 码块 CRC 附加（§5.2）
3. **信道编码**（§5.3）：数据用 **LDPC**，控制用 **Polar**（小信息位用 RM/重复/simplex 等小块码）
4. 速率匹配（§5.4）
5. 码块级联（§5.5）

## 上行传输信道与控制信息（§6）
- 随机接入信道（§6.1）
- **UL-SCH**（§6.2，LDPC）
- **UCI**（§6.3）：HARQ-ACK、SR、CSI（Polar / 小块码）

## 下行传输信道与控制信息（§7）
- **BCH**（§7.1，Polar）
- **DL-SCH / PCH**（§7.2，LDPC）
- **DCI**（§7.3，Polar）：各 DCI 格式（0_0/0_1 上行调度，1_0/1_1 下行调度，2_x 组公共等）
- Wake-up information（§7.4）

## 关键结论
- **LDPC**：UL-SCH / DL-SCH（两套 base graph：BG1 大块、BG2 小块）。
- **Polar**：BCH / DCI / UCI（信息位 > 11 bit）。
- 映射到物理信道见 [[TS38.211_PHY-Channels]]（§4）。
