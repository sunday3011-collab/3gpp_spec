# TS38.213 — Physical layer procedures for control

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.213）。
> 原始文档：[[38.213_PHY-Control_j30]]（`raw_sources/specs/TS38.213_PHY_Procedures/38.213_PHY-Control_j30.md`）

- **层**：PHY（RAN WG1）
- **职责**：控制相关物理层过程——同步、RLM、波束恢复、上行功控、随机接入、PDCCH 监测、BWP 操作等。

## 主要过程（§ 顶层）

| clause | 过程 | 要点 |
|--------|------|------|
| §4 | 同步 | 小区搜索（PSS/SSS/PBCH）、发送定时调整（TA）、SCell 激活定时 |
| §5 | **无线链路监测 RLM** | in-sync/out-of-sync 指示 → 配合 RRC 的 N310/N311/T310（[[TS38.331_RRC]]） |
| §6 | 链路恢复 | 波束失败检测与恢复（BFR） |
| §7 | **上行功控** | PUSCH/PUCCH/SRS/PRACH 功率；PHR（§7.7） |
| §8 | **随机接入过程** | 前导、RAR（Type-1 4-step / Type-2 2-step）、RAR UL grant 调度的 Msg3 PUSCH |
| §9 | 发送控制信息 | UCI（HARQ-ACK/SR/CSI）在 PUCCH/PUSCH 的复用 |
| §10 | 接收控制信息 | **PDCCH 监测**：CORESET、search space、DCI 检测 |
| §11 | UE-group 公共信令 | 组公共 DCI（SFI、抢占、TPC 等） |
| §12 | BWP 操作 | 激活 BWP、bwp 切换 |
| §13 | Type0-PDCCH CSS | SIB1 调度监测 |

> 其余：§14 IAB、§15 DAPS HO、§16 sidelink、§17 RedCap、§18 MBS、§20 NCR、§21 L1/L2 触发移动性（LTM）等。

## 与协议栈的接口
- RLM（§5）out/in-sync 指示驱动 [[TS38.331_RRC]] 的 T310/T312。
- 随机接入（§8）配合 [[TS38.321_MAC]] §5.1 的 RA 过程（Msg1–Msg4 / MsgA–MsgB）。
- 功控（§7）与 MAC PHR（§5.4.6）联动。
