# PHY 层

**职责**：在空口上完成物理信号的编码、调制、资源映射与收发，向 MAC 提供传输信道（transport channel）。

- **对应 TS 规范**：TS38.211 Physical channels and modulation / TS38.212 Multiplexing and channel coding / TS38.213 Physical layer procedures for control / TS38.214 Physical layer procedures for data
- **WG 归属**：RAN WG1

## 接口关系
- **上层（MAC）**：通过**传输信道**向 [[MAC/README]] 提供数据传输服务。
- **下层**：空口（air interface），无下层协议。
