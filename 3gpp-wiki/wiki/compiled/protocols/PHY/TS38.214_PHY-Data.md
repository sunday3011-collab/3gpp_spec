# TS38.214 — Physical layer procedures for data

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS38.214）。
> 原始文档：[[38.214_PHY-Data_j30]]（`raw_sources/specs/TS38.214_PHY_Data/38.214_PHY-Data_j30.md`）

- **层**：PHY（RAN WG1）
- **职责**：数据信道（PDSCH/PUSCH）相关物理层过程——调度、MCS/TBS、CSI、参考信号、处理时间。

## 主要过程

| clause | 过程 | 要点 |
|--------|------|------|
| §4 | 功控 | 下行功率分配 |
| §5 | **PDSCH 相关** | 接收过程（§5.1）、**CSI 上报**（§5.2）、PDSCH 处理时间（§5.3）、CSI 计算时间（§5.4） |
| §6 | **PUSCH 相关** | 发送过程（§6.1）、RS（§6.2）、频率跳变（§6.3）、PUSCH 准备时间（§6.4） |
| §7 | intra-cell guard band 上收发 | |
| §8 | PSSCH 相关 | sidelink 数据信道过程 |

## 关键结论
- **MCS / 调制阶数 / TBS**：PDSCH 见 §5.1.3，PUSCH 见 §6.1.4；多张 MCS 表（含 256QAM、低 SE / URLLC 表）。
- **CSI 上报**（§5.2）：CQI、PMI、RI、CRI、LI；周期/半持续/非周期；resource setting 与 report setting。
- **资源分配**：时域（type A/B，SLIV）、频域（type 0 bitmap / type 1 RIV）。
- 处理时间能力：PDSCH 处理（§5.3，N1）、PUSCH 准备（§6.4，N2），与 UE capability、SCS 相关。

## 与协议栈的接口
- 调度由 [[TS38.321_MAC]] 经 DCI 下发（DCI 格式见 [[TS38.212_PHY-Coding]] §7.3）。
- 物理资源/信号定义见 [[TS38.211_PHY-Channels]]；控制过程见 [[TS38.213_PHY-Control]]。
