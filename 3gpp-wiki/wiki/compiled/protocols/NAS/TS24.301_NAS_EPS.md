# TS24.301 — NAS protocol for Evolved Packet System (EPS)

> 蒸馏页：常用结论 + 精确 clause 引用。深挖见 `wiki/sections.tsv`（spec=TS24.301）。
> 原始文档：[[24.301_NAS_EPS_j60]]（`raw_sources/specs/TS24.301_NAS_EPS/24.301_NAS_EPS_j60.md`）

- **层**：NAS（CT WG1）；UE ↔ **MME**（EPC）端到端。是 5GS NAS（[[TS24.501_NAS_5GS]]）的 4G/EPS 对应规范。
- 两大子层：**EMM**（EPS 移动管理）与 **ESM**（EPS 会话管理）。在 5G 研发中主要用于 **EPS Fallback / NSA(EN-DC) / 互操作** 场景。

## EMM（§5）
| 过程 | 要点 |
|------|------|
| **Attach** | `Attach Request`→`Attach Accept`→`Attach Complete`（附着 + 默认承载） |
| **TAU**（Tracking Area Update） | 移动性/周期更新 |
| **Service Request** | 从 EMM-IDLE 恢复连接 |
| Authentication / Security Mode | EPS AKA；NAS 安全激活 |
| Identity / GUTI Reallocation | 标识管理 |

- 状态：EMM-DEREGISTERED ↔ EMM-REGISTERED；EMM-IDLE / EMM-CONNECTED。标识：GUTI、S-TMSI、IMSI。

## ESM（§6）
| 过程 | 要点 |
|------|------|
| **PDN Connectivity** | 建立 PDN 连接（默认 EPS 承载） |
| Bearer Resource Allocation/Modification | 专用承载（QCI、TFT） |
| EPS Bearer Context Activation/Modification/Deactivation | 承载管理（网络发起） |

## 消息格式（§8/§9）
EMM/ESM 消息头与 IE 编码。
